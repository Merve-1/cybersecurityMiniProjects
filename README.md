# VMware + Kali Linux Setup Guide

This repository serves as a **step-by-step guide** for installing VMware Workstation/Fusion and setting up a Kali Linux virtual machine for personal use.

---

## 1. Download VMware
1. Go to the official VMware download page:  
   👉 https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion  
2. As shown in the image below, click **"Download Now"**.
      <img width="1877" height="895" alt="image" src="https://github.com/user-attachments/assets/41a2d9ad-4ae3-4326-9a55-00ba633b7019" />
3. You will be redirected to the VMware login page:  
   - If you already have an account → **Sign in**. <img width="1894" height="895" alt="image" src="https://github.com/user-attachments/assets/34fc7da0-66c5-45ba-874c-2d7f809adf85" />

   - If you are new → **Create a new account**.  <img width="1842" height="847" alt="image" src="https://github.com/user-attachments/assets/881cf985-e55d-40bf-98eb-c8c75d22992c" />

4. Download the VMware installer for **personal use** (select your OS: Windows, macOS, or Linux). <img width="1060" height="237" alt="image" src="https://github.com/user-attachments/assets/bcef5895-6c6f-4aba-96a1-49baa25a320c" />


---

## 2. Install VMware
- Run the downloaded installer.  
- After completion, open VMware Workstation (Windows/Linux) or VMware Fusion (macOS).

---

## 3. Download Kali Linux
1. Visit the official Kali Linux download page:  
   👉 https://www.kali.org/get-kali/
   <img width="1616" height="730" alt="image" src="https://github.com/user-attachments/assets/a60c9ef2-c54f-4b70-a903-9ce6814ada9b" />
2. Choose the **Virtual Machines** section.  
3. Download the image that matches your system (VMware version).

---

## 4. Import Kali Image into VMware
1. Open VMware.  
2. Navigate to **File → Open**.  
3. Select the Kali `.iso` file you downloaded.  
4. Follow the prompts to import the virtual machine.  

---

## 5. Use the Video Tutorial (Optional)
You can also follow this video walkthrough for downloading VMware, setting up the installer, getting the Kali image, and installing it:

---

## 6. Start Kali & Begin
- In VMware, select your Kali VM and click **"Power On"**.  
- Log in using the default credentials provided on the Kali site.  
- Begin exploring and using hacking tools for legal and ethical purposes only.
